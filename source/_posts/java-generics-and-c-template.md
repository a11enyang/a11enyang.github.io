# c++ 与 java 的相似之处

## 继承

子类的构造函数中都会先进入基类的构造函数进行运行

* java中需要使用`super（）进行显式访问`
* c++不需要



## 泛型

* c++中使用template<class T>的方式声明
* java中使用class A<T>的方式声明
* 两者声明的方式不同，但是使用的时候方法都是一样的ClassName<T>都是采用这种方式进行





## 库

* java 的collections
* c++的stl
